Iniciar a aplicação

Para iniciar a aplicação é necessário ter instalado no computador a JDK 8.

Também é necessário ter o mysql instalado na máquina, pois ele usa banco de dados para salvar os dados.

Precisa-se importar o banco de dados na última versão ( pasta BD da raiz do projeto)

Para iniciar a aplicação basta executar no terminal (na pasta dist da raiz do projeto):

java -jar "Atividade_Pratica_Supervisionada.jar"

Ou ir na pasta dist raiz do Projeto e dar 2(dois) clicks em "Atividade_Pratica_Supervisionada.jar"
